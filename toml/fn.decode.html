pub fn decode<T: Decodable>(toml: Value) -> Option<T>

Decodes a TOML value into a decodable type.

This function will consume the given TOML value and attempt to decode it into the type specified. If decoding fails, None will be returned. If a finer-grained error is desired, then it is recommended to use Decodable directly.
